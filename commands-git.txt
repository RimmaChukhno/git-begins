# Git cloning command:
	git clone repository address

## Three commands to push a file to GitHub:
	1. git add filename 
	2. git commit -m "Message text"
	3. git push origin <branch-name> 

### Command to pull a file:
	git pull <branch name>

#### Command to create a new branch and to switch to it:
	git checkout -b <new branch name>
	git switch <branch-name>
